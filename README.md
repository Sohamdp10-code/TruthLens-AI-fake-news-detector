## 🎯 About
TruthLens is an AI-powered fake news 
detector that analyzes news articles 
and provides instant credibility scores.

## ✨ Features
- 🤖 AI powered news analysis
- 📊 Credibility score (0-100%)
- 🚩 Red flags detection
- ✅ Credibility signals
- 🔑 Keyword highlighting
- 📰 Trusted sources list
- 📱 Responsive design
- 📥 Download analysis report
- 📤 Share results

## 🛠️ Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- NLP Algorithm
- Sentiment Analysis
